# Portafolio de Armando Monreal

Portafolio interactivo con temática espacial de Armando Monreal, desarrollador web Full Stack.

## Tecnologías

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Motion

## Ejecución local

1. Instala las dependencias con `npm install`.
2. Inicia el entorno de desarrollo con `npm run dev`.
3. Abre `http://localhost:3000`.

## Proyectos presentados

- El Rinconcito — aplicación Full Stack para un negocio de comida.
- Serenity Space — aplicación frontend de bienestar y relajación.
- GasLeak — sistema IoT para detectar fugas de gas y automatizar la ventilación.
